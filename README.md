# تطبيق رحلة البيانات: من الصورة إلى الصوت 🚀

هذا التطبيق هو مختبر تعليمي تفاعلي تم بناؤه باستخدام **Streamlit** و **OpenCV** و **EasyOCR** و **gTTS**. يوضح التطبيق رحلة البيانات من بكسلات الصورة إلى كلمات مسموعة مع شرح تقني وبرمجي لكل مرحلة.

## 🛠️ كيفية التشغيل محلياً

1. **تثبيت المتطلبات**:
   تأكد من تثبيت Python على جهازك، ثم قم بتشغيل الأمر التالي في التيرمينال:
   ```bash
   pip install -r requirements.txt
   ```

2. **تشغيل التطبيق**:
   قم بتشغيل الأمر التالي لبدء التطبيق:
   ```bash
   streamlit run app.py
   ```

## 🌟 الميزات المضمنة
- **المعالجة البصرية المعمقة**: شرح Grayscale و Thresholding.
- **محرك OCR**: رسم مربعات الكشف وشرح الخوارزمية.
- **التوليد الصوتي التفاعلي**: أنيميشن وكود حي متزامن.

أتمنى لك كل التوفيق في عرضك الجامعي!
